Horizon Suite — custom class icons
==================================

Place one square texture per class (recommended 32x32 or 64x64). Format: TGA.

Path pattern:
  media/CustomClassIcons/<CLASSFILE>/<classfile lower>.tga

Examples:
  WARRIOR/warrior.tga
  DEATHKNIGHT/deathknight.tga
  DEMONHUNTER/demonhunter.tga

Folder names are uppercase (same as UnitClass classFile). File names are lowercase .tga.

In-game: Horizon options → Class icon style → Custom (addon media), then /reload.

Classes: DEATHKNIGHT, DEMONHUNTER, DRUID, EVOKER, HUNTER, MAGE, MONK,
          PALADIN, PRIEST, ROGUE, SHAMAN, WARLOCK, WARRIOR
