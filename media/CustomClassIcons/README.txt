Horizon Suite — custom class icons
==================================

Place one square texture per class (32x32–128x128 is fine; e.g. 120x120 scales down cleanly). Format: BLP.

On-screen draw is slightly larger than Default/Rondo when Horizon (bundled icons) is selected (tooltip ~20px design, dashboard ~38px design, before UI scale).

Path pattern:
  media/CustomClassIcons/<CLASSFILE>/<classfile lower>.blp

Examples:
  WARRIOR/warrior.blp
  DEATHKNIGHT/deathknight.blp
  DEMONHUNTER/demonhunter.blp

Folder names are uppercase (same as UnitClass classFile). File names are lowercase .blp.

In-game: Horizon options → Class icon style → Horizon, then /reload.

Classes: DEATHKNIGHT, DEMONHUNTER, DRUID, EVOKER, HUNTER, MAGE, MONK,
          PALADIN, PRIEST, ROGUE, SHAMAN, WARLOCK, WARRIOR
