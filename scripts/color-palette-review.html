5. Semantic Color Grouping — Start From Scratch

Principle: Categories that are conceptually related should share a color family. Similar colours = related content. Fresh palette — inspired by WoW in-game UI, not derived from existing Config.lua.

RAID uses green to match the in-game raid icon. All colors chosen independently; Blizzard references where the game establishes them (campaign gold, recurring cyan, Epic purple, etc.).
